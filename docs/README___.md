# Документация по проектной (учебной) практике

## Общая информация

**Проект:** «КупиУмно»

**Цель проекта:** разработка концепции интеллектуальной платформы для анализа товаров и цен на маркетплейсах.

В рамках практики были изучены основы работы с Git, синтаксис Markdown и технологии создания статических веб-сайтов с использованием HTML и CSS.

---

## Этап 1. Работа с Git

В ходе практики был создан репозиторий проекта на GitHub.

Освоены основные операции системы контроля версий Git:

* git clone
* git add
* git commit
* git push
* git branch
* git checkout

Изменения в проекте регулярно фиксировались с помощью коммитов с осмысленными сообщениями.

---

## Этап 2. Изучение Markdown

Был изучен синтаксис языка разметки Markdown.

Получены навыки оформления:

* заголовков различных уровней;
* списков;
* таблиц;
* ссылок;
* блоков кода;
* изображений.

Markdown использовался для подготовки документации проекта и оформления материалов практики.

---

## Этап 3. Разработка сайта

Создан статический веб-сайт проекта «КупиУмно» с использованием HTML и CSS.

Структура сайта включает следующие страницы:

* Главная страница;
* О проекте;
* Участники;
* Журнал проекта;
* Ресурсы.

Для всех страниц реализована единая система навигации.

---

## Этап 4. Наполнение контентом

На сайт добавлены:

* описание проекта;
* информация об участниках команды;
* журнал выполнения работ;
* список полезных ресурсов;
* графические материалы.

---

## Этап 5. Тестирование и доработка

Выполнена проверка:

* корректности ссылок между страницами;
* структуры проекта;
* отображения контента;
* оформления пользовательского интерфейса.

После проверки были устранены выявленные недочёты.

---

## Результат

В результате выполнения практики был разработан статический веб-сайт проекта «КупиУмно», оформлена документация в формате Markdown и подготовлены материалы для итогового отчёта.
