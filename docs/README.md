# Документация по проектной (учебной) практике

## Общая информация

**Проект:** «КупиУмно»

**Цели и задачи проекта:** создание сервиса, позволяющего пользователям эффективно анализировать предложения различных интернет-магазинов и маркетплейсов. Платформа предоставляет инструменты для сравнения цен, отслеживания динамики стоимости товаров, оценки реальности скидок и получения рекомендаций по наиболее выгодному моменту совершения покупки. Реализация проекта позволит сделать процесс онлайн-шопинга более удобным, информированным и экономически выгодным для пользователей.

В ходе выполнения практики были освоены базовые технологии и инструменты, необходимые для реализации веб-проектов. Изучены возможности системы контроля версий Git, обеспечивающей совместную разработку и контроль изменений исходного кода. Рассмотрен синтаксис языка разметки Markdown и его применение для подготовки проектной документации. Кроме того, получены практические навыки разработки статических веб-сайтов средствами HTML и CSS, включая создание структуры страниц, оформление интерфейса, работу с навигацией и организацию контента. Полученные знания были применены при создании сайта и подготовке сопроводительной документации.

---

## журнал прогресса

### Этап 1. Работа с Git

В ходе практики был создан репозиторий проекта на GitHub.

Освоены основные операции системы контроля версий Git:

* git clone
* git add
* git commit
* git push
* git branch
* git checkout

Изменения в проекте регулярно фиксировались с помощью коммитов с осмысленными сообщениями.

---

### Этап 2. Изучение Markdown

Был изучен синтаксис языка разметки Markdown на [портале](https://ru.hexlet.io/lesson_filters/markdown).

Получены навыки оформления:

* заголовков различных уровней;
* списков;
* таблиц;
* ссылок;
* блоков кода;
* изображений.

Markdown использовался для подготовки документации проекта и оформления материалов практики.

---

### Этап 3. Разработка сайта

Создан статический веб-сайт проекта «КупиУмно» с использованием HTML и CSS.

Структура сайта включает следующие страницы:

* Главная страница;
* О проекте;
* Участники;
* Журнал проекта;
* Ресурсы.

Для всех страниц реализована единая система навигации.

---

### Этап 4. Наполнение контентом

На сайт добавлены:

* описание проекта;
* информация об участниках команды;
* журнал выполнения работ;
* список полезных ресурсов;
* графические материалы.

---

### Этап 5. Тестирование и доработка

Выполнена проверка:

* корректности ссылок между страницами;
* структуры проекта;
* отображения контента;
* оформления пользовательского интерфейса.

После проверки были устранены выявленные недочёты.

---

## Результат

В результате выполнения проектной (учебной) практики были освоены базовые инструменты и технологии, применяемые при разработке веб-проектов. Изучены принципы работы с системой контроля версий Git, освоен синтаксис языка разметки Markdown и получены практические навыки создания статических веб-сайтов с использованием HTML и CSS.

В ходе практики был разработан статический веб-сайт проекта «КупиУмно», включающий главную страницу, разделы с описанием проекта, информацией об участниках, журналом выполнения работ и подборкой полезных ресурсов. Для сайта была реализована единая структура навигации и выполнено оформление пользовательского интерфейса.

Помимо разработки сайта была подготовлена документация проекта в формате Markdown, отражающая основные этапы выполнения практики и результаты проделанной работы. Также были собраны и оформлены материалы, необходимые для подготовки итогового отчёта по практике.

Таким образом, поставленные задачи практики были выполнены в полном объёме, а полученные знания и навыки могут быть использованы при дальнейшей разработке веб-приложений и работе над командными проектами.
